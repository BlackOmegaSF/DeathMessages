# Death Messages

## Modified Death Messages

Death messages in the server chat show the death location coordinates. Click the coordinates to teleport.

When a player is killed by any entity using a Diamond Sword named `The Bitchwhipper`, a custom death message is shown.

## Custom Sounds

When a player dies, the custom sound `kleinercode\oof` is played.

When a player is killed by `The Bitchwhipper`, the custom sound `kleinercode\whip` is played.

When any entity is killed by a Diamond Sword with the name `The Bitchripper`, the custom sound `kleinercode\whip` is played.

These custom sounds come from my resource pack: [ServerResourcePack](https://github.com/BlackOmegaSF/ServerResourcePack)
